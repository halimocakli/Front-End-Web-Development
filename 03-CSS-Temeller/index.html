WEB'in Temel Dili: HTML

Tüm modern WEB siteleri ve WEB uygulamaları üç temel teknoloji kullanılarak oluşturulur: HTML, CSS ve JavaScript. Bunlar WEB'in dilleridir.

Bu yazıda HTML'ye odaklanacağız ve HTML'nin ne olduğunu ve neden sizin de HTML öğrenmeniz gerektiğini öğreneceğiz.

HTML Nedir?

HTML, HyperText Markup Language (Bağlantılı Metin İşaretleme Dili) anlamına gelir. Web geliştiricilerinin bir web sayfasının içeriğini yapılandırmak ve tanımlamak için kullandıkları bir biçimlendirme dilidir (bir programlama dili değildir). HTML, farklı içerik türlerini tanımlayan öğelerden oluşur: paragraflar, bağlantılar, başlıklar, resimler, videolar, vb. WEB tarayıcıları HTML'yi anlar ve HTML kodunu web siteleri olarak işler.

HTML, farklı içerik türlerini tanımlayan şu öğelerden oluşur: paragraflar, bağlantılar, başlıklar, resimler, videolar, vb. WEB tarayıcıları, HTML'i anlar ve HTML kodunu WEB siteleri haline getirecek şekilde işler.

HTML'de herbir element 3 bölümden oluşur:

  1. Açılış Etiketi
  2. Kapanış Etiketi
  3. Ana Element

MDN Web Dokümanları üzerinden daha fazla bilgi edinebilirsiniz.

Neden HTML Öğrenmelisiniz?

WEB'in temel dilini öğrenmek için sayısız neden mevcuttur. İşte bu sebeplerden 5 tanesi şu şekildedir:

  • Temel WEB geliştirme dilini kullanabilme.
  • Worpress veya Wix gibi araçlara güvenmek yerine güzel WEB sitelerini kendiniz oluştururabilmek için
  • Kapsamlı WEB uygulamaları oluşturmak için
  • Arkadaşlarınızı etkilemek için
  • Eğlenmek için 😃
